## Welcome to This Custom Extension Gallery
This repository is a fork of SharkPool’s Extension Collection, adapted into a custom gallery that uses the same structure, layout, and interface as the original. It serves as a place to host and display extensions in a clean and lightweight format.

Please don’t copy or redistribute extensions from this gallery into other galleries without permission from their respective creators.
